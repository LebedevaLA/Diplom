1) Сначала запустили GenerateFunc_1 - сгенерировали по 25 функций из каждого класса (3 класса) для тренировки CVAE модели и для запуска функции удаления,  
сгенерированные функции находятся в train_functions.pkl, графики этих функций в папке StartFunctionsPlots(25*3)  
  
2) Запустили FindParams_Delete_2 - для каждой функции из train_functions.pkl нашли гиперпараметры и результаты функции оптимизации  
с удалением точек, 5% лучших результатов после удаления для каждой функции можно посмотреть в папке del_results, в этой папке в файле all_results.pkl  
сохранены все данные о функции + результаты(параметры) без удаления + лучший результат(параметры) с удалением + гиперпараметры  
  
3) Запустили Delete_Analysis_3 - для каждой функции рисуем 4 графика на одной картинке, первый график - функция исходная,  
   второй функция без удаления с подобранными оптимизатором параметрами + исходная, третий график - функция с параметрами, подобранными с удалением + исходная,  
   четверный график - функция с параметрами, подобранными с удалением + точки которые остались после удаления. Эти графики находятся в папке comparison_plots_res.  
   Так же в Delete_Analysis_3 вычисляются средние гиперпараметры для каждого классаи сохраняются в Mean_HyperP.pkl
  
4) Запустили Create_TestFunc_4 - генерируем данные для тестирования CVAE модели, добавляем немного функций из новых классовс более сложными функциями.  
   Данные о тестовых функциях находятся в train_functions.pkl. Графики тестовых функций находятся в папке test_plots
  
5) Запустили CVAE_5 - создание, обучение и тестирование модели, создает файлы cvae_weights.pkl, scaller.pkl, картинки - test_confusion_matrix.png и training_history.png,  
   Так же создает correctly_classified.pkl - данные о верно классифицированных тестовых функциях
